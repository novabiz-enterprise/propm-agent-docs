## 7. Contingency Drawdown Rules

- Contingency can be used only for approved and justified uncertainty events.
- Drawdown approval up to USD 150,000 by Project Director.
- Drawdown above USD 150,000 requires CCB approval.
- Any drawdown above USD 500,000 requires Steering Committee approval.
