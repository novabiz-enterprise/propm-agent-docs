---
title: What is ProPM Agent?
---

ProPM Agent is a web application for running project delivery with **shared knowledge**, **AI-assisted execution**, and **traceable governance evidence**.

At a high level, you:

1. **Create or open a project**.
2. **Upload and classify documents** in Knowledge.
3. **Run agents** (orchestrator + specialists) to draft and analyze project content.
4. **Review and edit PM Docs** with exportable outputs.
5. **Track runs/activity** in an audit-friendly lifecycle.

Typical navigation flow:

- Dashboard → Portfolio Command Center → Project Workspace → Agents/Knowledge/PM Docs → AI Log

For demonstrations, the product includes a seeded sample project:

- **Azure Bay Hotel & Convention Center** (`demo-hotel-001`)
- Includes charter, stakeholder register, WBS, schedule, budget, RAID, risk, change control, and procurement artifacts.

## Designed for enterprise deployment

ProPM Agent is typically deployed from **Azure Marketplace** as a **Managed Application** into your organization’s Azure subscription.

This approach supports enterprise expectations such as tenant isolation, consistent deployment, and alignment with security review processes.
