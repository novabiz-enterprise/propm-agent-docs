## 2026-03-15

- **Release date:** 2026-03-15
- **Highlights:** richer default demo experience with built-in project import/delete lifecycle and seeded walkthrough content
- **New capabilities:**
  - import the default demo project from the **Projects** page
  - delete and later re-import the default demo project
  - default demo project grants full rights and permissions to the signed-in user
  - seeded AI Log runs and activity history in the demo project
- **Improvements:**
  - expanded demo Knowledge and PM Docs coverage
  - expanded hotel-construction demo input pack with communication plan, weekly status report sample, and app test scenarios
  - updated user documentation to describe demo-ready behavior across Projects, Workspace, Knowledge, PM Docs, Agents, AI Log, and Access Control
- **Fixes:**
  - aligned demo project categories with procurement and communication management artifacts
- **Known limitations:** see the Known limitations page

## Format

For each release, include:

- **Release date**
- **Highlights** (business value)
- **New capabilities**
- **Improvements**
- **Fixes**
- **Known limitations** (link to the Known limitations page)
